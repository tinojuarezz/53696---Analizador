Para probar la validez de los inputs, debe dirigirse al archivo input.txt (dentro del visualstudiocode) y colocar una de las cadenas otorgardas en el repositorio (con el nombre de inputcorrecto/inputincorrecto). 
Una vez copiada la cadenas, guarde el archivo (ctrl + s) e ingrese a la terminal. (Instale las dependencias en caso de no tenerlas instaladas), Luego copie el siguiente codigo con el orden dado: 

1. cd "C:\Users\Usuario\ssl-antlr-calculator"
2. npm.cmd start

En caso de querer probar otra cadena, copie una diferente, vuelva a guardar el archivo y en la terminal ejecute nuevamente: 

npm.cmd start
